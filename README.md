# practic_project
## Проектная практика

Мной выбрана vision модель для классификации состояния растений по изображению листа

Цель - определить является ли растение здоровым или поражённым болезнью по изображению листа, это помогает
сократить время ручного осмотра, выявить проблемы с растениями и минимизировать потери урожая

Датасет - https://www.kaggle.com/datasets/emmarex/plantdisease/  (нужно положить в корень директории с файлом project.ipynb)  
Модель - https://huggingface.co/Carina124/plant-vit-model-3      (скачивается автоматически при запуске jupiter-notebook)

Playbook был написан так, что он рекурсивно ходит в папку PlantVillage и считывает файлы по очереди.  
При необходимости можно изменить название директории для датасета -
```root_folder = "PlantVillage"```
